Ever wanted your raft to craft for you while you focus on building and exploring? That is exactly what AutoCrafter does.

Important: AutoCrafter is currently an alpha version.
Do not trust it 100 percent for critical worlds yet. Bugs are always possible.

**What it does:**
Upgrade normal storage chests into AutoCrafters with up to 3 recipe slots. Each slot can run continuously or with a fixed count, and can optionally pull ingredients from a dedicated input chest and send crafted items to a dedicated output chest.

**Features:**
- Upgrade any chest to AutoCrafter level 1-3
- Up to 3 independent crafting slots
- Infinite mode or count-limited mode per slot
- Optional input and output container assignment per slot
- Host-side crafting loop for multiplayer safety
- Per-world save data for crafter state
- Works with AutoSorter mod in the current version

**Quick Start:**
1. Place a small storage chest.
2. Open it and upgrade it in the AutoCrafter panel.
3. Pick a recipe for slot 1.
4. Add ingredients to input inventory.
5. Activate the slot and wait for the craft tick.

**Example Use Case (Aloe Vera):**
- Chest A: contains clay and palm leaves (input)
- Chest B: upgraded AutoCrafter chest with Aloe Vera recipe
- Chest C: receives crafted Aloe Vera (output)

AutoCrafter pulls from Chest A, crafts in Chest B, and places results in Chest C.
